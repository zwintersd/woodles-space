A Block System for Textbook-Style Pages

✦

Not an app you click around in — a system you write with. Give any capable model the voice spec below and it writes in a consistent register; give it the chrome and it drops that writing into a page that looks like every other page here; give it the example blocks and its worked examples get a real diagram instead of a wall of description. Three layers, one look, no proprietary tooling required — it's all plain HTML, CSS, and SVG, readable straight from the source of any page in this library.

1

How the Three Layers Fit

The content spec shapes the writing. The chrome wraps it in the same masthead, headers, and callouts every time. The example blocks give topic-specific worked examples — a mechanism, a comparison, a timeline, a chart — an actual visual shape instead of another paragraph.

2

The Library

Two reference files, documented and live-rendered so a fresh model session can read the source and replicate the classes exactly — plus worked pages proving the pipeline end to end on real material, one light topic and one heavy one.
